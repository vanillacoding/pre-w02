# JavaScript Homework

이번 주에는 드디어 자바스크립트를 이용해 코드를 작성해보려고 합니다. 🎉

HTML, CSS는 잠시 내려두고 우리 자바스크립트에 집중해보는 시간을 가져봅시다.

이번 주 우리는 TDD 방식이 적용된 과제를 진행해보려고 하는데요.

[TDD(Test-Driven Development)](https://ko.wikipedia.org/wiki/%ED%85%8C%EC%8A%A4%ED%8A%B8_%EC%A3%BC%EB%8F%84_%EA%B0%9C%EB%B0%9C)란 테스트 주도 개발이라는 의미입니다. 하나의 제품을 모두 완성 후 테스트를 진행하는 방식이 아니라, TDD는 소프트웨어가 완전히 개발되기 전에 소프트웨어 요구 사항을 테스트 케이스로 작성하고, 모든 테스트 케이스에 대해 반복적으로 테스트하여 모든 소프트웨어 개발을 추적하는 소프트웨어 개발 프로세스입니다.

## 과제 진행 방법
homeworks 아래 5가지의 `js` 파일은 각각 같은 이름의 html 파일과 연결되어 있습니다.

`js` 파일에 코드를 작성하고, 해당 이름의 html 파일을 브라우저에서 열면 테스트 결과를 확인할 수 있는 방식인데요. ([VSCODE에서 HTML 파일 쉽게 브라우저로 열기](https://www.youtube.com/watch?v=T8DrI6K8ArE&ab_channel=AlanSimpson)를 참고해보세요.)

`1_practice.js` 파일에 어떻게 테스트를 실행시키고 결과를 확인할 수 있는지 가이드가 준비되어 있으니, 아래 순서에 따라 과제를 시작해보세요.

1. `index.html` 폴더를 브라우저에서 열어주세요.
2. `1. 연습게임` 박스를 눌러보세요.
3. 코드 에디터에서 homeworks/1_practice.js 파일을 열어주세요.
4. 테스트를 통과할 수 있도록 코드를 수정해주세요.
5. 코드 작성 후 vscode에서 저장 후, 브라우저에서 새로고침을 눌러 테스트 결과를 확인해보세요.

이 과정을 반복하며 5개의 과제를 모두 완성해주시면 됩니다!

> <b>✨ 사소하지만, 유용한 팁</b>
- vscode에서 파일을 저장 단축키 (`cmd + s`, `ctrl + s`)
- 브라우저에서 페이지 새로고침 단축키 (`cmd + r`, `ctrl + r`)
